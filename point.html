Pickeable Point

Pickeable Point

It is a point with a feedback when the mouse is over it. The feedback can be overriden by the user. By default, the feedback is that the point grows in size when the mouse is over it.

Simple Point


import java.awt.Color;
import processing.core.PApplet;
import util.VAPoint;

public class PointExample extends PApplet{

 private VAPoint point;
 public void setup()
 {
 point = new VAPoint(250, 250, this);
 size(500, 500);
 }
 
 public void draw()
 {
 if(point.mouseIsOverFeedback())
 {
 background(Color.white.getRGB());
 }
 point.drawPoint();
 }
}
