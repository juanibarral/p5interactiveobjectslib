Simple library over processing for visual analytics

Welcome to P5 Visual Analytics Library.

This is a a library written in Java over Processing for visual analytics. The main feature is that the elements have their own feedback and are optimized to be drawn only when necessary.

Visual Elements

There are simple elements to use as wel as complete plots with whole funcionality.

  • Point
  • Parallel Coordinates Plot
  • Dandelion Plot
